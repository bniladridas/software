How to Set Up Your API Key

Navigate to your account settings and locate the API section.

Generate a new API key. Make sure to save this key in a secure location as it will only be shown once.

Enter Your Together AI API Key

Your API key will be used only for making requests to the Llama-4-Maverick model and will not be stored on our servers.

Note: Your API key will be stored locally in your browser's storage and will only be used to make requests to the Together AI API. We do not store or transmit your API key to our servers.

Why Use Your Own API Key?

Using your own API key provides several benefits:

  • Higher rate limits tailored to your needs
  • Access to the latest model versions as they become available
  • Enhanced privacy as your requests go directly from your browser to Together AI
